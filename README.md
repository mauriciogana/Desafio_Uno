1) Descargar la rama Desafio_Uno_mgana
2) Crear la carpeta "c:/salida" donde quedará el archivo de salida

3) Opciones de ejecución:
	3.1) Desde eclipse ejecutar clase "com.previred.operacion.Launcher" (click derecho -> run as -> java application)
	3.2) Exportar como Runnable jar (click derecho -> export -> Runnable jar -> seleccionar: Launcher - Desafio_Uno)
	
4) El archivo con la entrada y la salida json se encuentran dentro del proyecto
